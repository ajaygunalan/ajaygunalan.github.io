# :boom: [My Professional Portfolio Website](https://ajaygunalan.github.io/):exclamation:

For build and local-host run with google analytical:

    JEKYLL_ENV=production jekyll serve 
   or
   
    JEKYLL_ENV=production jekyll build

For build and local-host run without google analytical:

    jekyll serve 
   or
   
    jekyll build
    
**Note**: I recommend always to use the latest release.
